# Выводы аудита
Аудировалась статья Михеевой "Особенности соотношения стрессоустойчивости и тревожности у мужчин и женщин" (2016) о связи стрессоустойчивости и тревожности у 291 респондента (149 мужчин и 142 женщин).

Что нашли:
По прозрачности всё плохо. Нет пререгистрации, нет открытых данных, нет кода, нет препринта. Автор не объяснила, как из сырых баллов получались уровни стрессоустойчивости и почему из анализа убрали группу со средними показателями.
По воспроизводимости тоже провал. Автор заявляет, что во всех подгруппах корреляция значима (p < 0,05), но не приводит ни одного конкретного значения r. Нет t-статистик, нет степеней свободы. Проверить математически невозможно.


Подозрительные моменты:
Исключение средней группы без объяснения причин - похоже на выборочное удаление данных. Подгрупповой анализ по полу и уровню стрессоустойчивости не заявлен изначально - признак HARKing. Восемь статистических тестов без коррекции уровня значимости - повышенный риск ложных находок.
